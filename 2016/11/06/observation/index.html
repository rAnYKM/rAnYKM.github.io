Social Networks, Communities and Node Attributes - Observations from Facebook Ego Networks

Community Detection is among the most popular research topics on social network. According to the definition, the nodes in the same community (module) have dense connections with each other. Many community detection algorithms or models originate from this idea. Among them, the Stochastic Block Model (SBM) is a generative graph model to produce graphs containing communities, subsets characterized by being connected with one another with particular edge densities. In this report, we conduct several experiments and discuss about the relationship among communities, network charateristics and attributes.

Network Density and Community Structure

Network Density is defined as the probability that one node connects with another one. Numerically, we can calculate it as
$$ \rho = \frac{|E|}{|V|*(|V|-1)}. $$

And in the undirected network/graph, we can calculate it as
$$ \rho = \frac{2|E|}{|V|*(|V|-1)}. $$

Wikipedia: Community Structure
In the study of complex networks sets of nodes such that each set of nodes is densely connected internally.

One common property of communities is that the network density is high inside the communities. In the experiment, we first select one attribute, then extract all the nodes with this attribute to build a sub-graph, and finally measure the network density of the sub-graph for comparison.

Clustering Coefficient

Clustering Coefficient is a measure of the degree to which nodes in a graph tend to cluster together. A high network density does not suggest a high clustering coefficient. There are two versions of clustering coefficient, the global and the local respectively.

Global Clustering Coefficient is defined as the number of closed triplets (or 3 * the number of triangles) over the total number of triplets (both open and closed), where a triplet consist of 3 connected nodes.

Local Clustering Coefficient quantifies how close a nodeâ€™s neighbours are to being a clique (complete graph). The LCC of node $u$ in the directed network can be calculated as follows.
$$ LCC_u = \frac{|(v,w):(v,w) \in V|}{k_u*(k_u-1)}, $$
where $k_u$ is the number of neighbors of node $u$.
